# Text File Analyzer

This is a simple Node.js project where I created a command line tool to analyze a text file.

In this project, I learned how to use the file system module (fs) to read files and get file details. I also used the path module to work with file names and paths.

The program takes a .txt file as input and calculates the number of lines, words, and characters in the file. It also shows the file size in a readable format.

I added different options like summary and detailed view, and also handled errors such as invalid file type and missing file.

This project helped me understand how to work with files in Node.js, handle command line arguments, and build simple backend utilities.
