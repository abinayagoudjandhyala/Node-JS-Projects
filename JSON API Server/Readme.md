# Node.js Backend API

In this project, I learned how to build a simple backend API using Node.js without using any frameworks. I understood how to create a server using the http module and handle different routes like users and posts.

I also worked with HTTP methods like GET and learned how to read data from JSON files using the file system module. I used this data to create API endpoints and display it in the browser.

This project helped me understand how URLs, query parameters, and ID-based routing work in real-world applications. It also gave me a clear idea of how backend systems manage and serve data.
