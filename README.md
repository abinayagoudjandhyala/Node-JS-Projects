# Node JS Projects

This repository contains the Node.js projects I built while learning backend development. I created these projects to understand how server-side programming works using JavaScript.

In this repository, I worked on different small projects like API creation, file handling, and command line tools. These projects helped me understand how Node.js works without using any frameworks.

By doing these projects, I learned how to use built-in modules like http, fs, and path. I also understood how to handle requests, work with files, and take input from the command line.

I practiced concepts like routing, reading JSON data, handling errors, and working with asynchronous operations. I also learned how backend systems process data and send responses.

Overall, these projects helped me build a strong foundation in Node.js and understand how backend applications are developed.
