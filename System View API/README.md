# System View API 

In this project, I built a simple API using Node.js to get system information like CPU, memory, operating system, user, network, and process details. I used core Node.js modules such as http, os, process, and url instead of any frameworks, which helped me understand how Node.js works internally. I created a server, handled routing manually, and returned responses in JSON format. I also implemented functions to format memory and uptime into a readable form. Through this project, I improved my understanding of backend development and how APIs work.
