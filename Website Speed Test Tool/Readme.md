# Website Speed Tester

This is a simple Node.js project where I built a command line tool to check the response time of websites.

In this project, I learned how to use the built-in http module in Node.js to send requests to websites and receive responses. I also understood how to calculate response time using Date and display important details like status code and connection time.

I handled different cases such as successful connections, errors, and timeouts. I also learned how to take input from the command line using process.argv and run the program for multiple websites.

This project helped me understand how backend systems communicate with servers, how HTTP requests work, and how real-world applications measure performance.
